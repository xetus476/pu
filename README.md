# MoneyApp

Этот репозиторий содержит набор учебных и экспериментальных проектов по машинному обучению, компьютерному зрению и глубокому обучению. В нём собраны Jupyter Notebook'ы, примеры работы с PyTorch, GAN, автоэнкодерами, YOLO, MediaPipe и базовыми моделями трансформеров.

## Что находится в репозитории

- Dl_begin — стартовые материалы по анализу данных и базовым задачам машинного обучения.
- itsc — эксперименты с глубоким обучением:
  - auto_encoder — автоэнкодеры и восстановление данных
  - gan — базовые GAN и WGAN
  - resnet — примеры с ResNet
  - cifri24.03 — работы с CIFAR-10 и генеративными моделями
  - stepin_s_encoderom — эксперименты с кодированием и признаками
  - vgg_+_stilization — стилизация изображений и эксперименты на базе VGG
- mediapipe_tutorial — примеры работы с MediaPipe для анализа изображений и видео.
- Yolo_knife — полноценный пример проекта по детекции объектов с использованием YOLO: обучение собственной модели, запуск инференса на изображениях и видео, работа с веб-камерой и интеграция с FastAPI для получения предсказаний через API.
- datasets — наборы данных для локальных экспериментов.
- vk_educ — учебные ноутбуки по PyTorch и трансформерам.
- mb — локальная виртуальная среда Python.

## Технологический стек

Проект использует:

- Python 3.12+
- Poetry
- PyTorch и torchvision
- scikit-learn
- OpenCV
- MediaPipe
- Ultralytics YOLO
- matplotlib, seaborn, pandas

## Установка

Рекомендуется использовать Poetry:

```bash
poetry install
```

Если вы используете локальную виртуальную среду, можно активировать её вручную:

```bash
# Windows
mb\Scripts\activate
```

## Как начать

1. Откройте нужный Jupyter Notebook.
2. Запустите ячейки по порядку.
3. Для задач компьютерного зрения используйте папки mediapipe_tutorial и Yolo_knife.
4. Для экспериментов с нейросетями — папку itsc.

### Yolo_knife: что здесь есть

В папке Yolo_knife собран практический пример работы с YOLO для обнаружения объектов. Здесь можно:

- обучить собственную модель на подготовленных данных;
- использовать готовые веса yolov8n.pt или yolo26n.pt;
- запускать инференс на изображениях и видео;
- подключать поток с веб-камеры для онлайн-детекции;
- использовать ноутбуки с примерами для обучения и inference;
- посмотреть вариант взаимодействия через FastAPI, когда модель доступна как веб-сервис.

Этот раздел особенно полезен, если нужно быстро перейти от обучения модели к её запуску в реальном времени или через API.

## Примечания

- Некоторые ноутбуки являются исследовательскими и могут требовать GPU.
- Часть данных и весов моделей хранится локально в репозитории или ожидается в соответствующих папках.
- Репозиторий скорее представляет собой набор учебных экспериментов, чем законченный продукт.

## Статус проекта

Проект находится в стадии экспериментов и обучения, с акцентом на исследование различных подходов в глубоком обучении и компьютерном зрении.
